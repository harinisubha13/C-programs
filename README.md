# C Programming

This repository contains basic C programs for learning and practice.

## Programs Included

### 1. Calculator (Calculator.c)
- Performs basic arithmetic operations
- Concepts used: Operators, Switch Case

### 2. Fibonacci Series (Fibonacci.c)
- Prints Fibonacci sequence
- Concepts used: Loops

### 3. Prime Number (Prime.c)
- Checks whether a number is prime or not
- Concepts used: Loops, Conditions
### 4. Student Management System (File Handling)
- Add Student
- Display Students
- Search Student
- Delete Student
- Concepts used: Structures, File Handling, Menu Driven Programming

## How to Run

Compile using:
gcc filename.c -o output

Run:
./output
